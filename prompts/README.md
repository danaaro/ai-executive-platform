# prompts/

Cross-product prompts ONLY. Product-specific prompts live under `products/<product>/prompts/`.

- `system/` — platform-level system prompts (guardrails, persona baseline, safety)
- `shared/` — reusable prompt fragments (tone, formatting, evidence rules)

Prompts are treated as versioned artifacts (see `../../foundation/26-prompt-generation.md` seed: base template + injection).
