# Job Description Agent — Question Bank

> STATUS: 🟢 v2 (2026-07-18) — **Job Discovery Questionnaire**, imported from Susan's production attachment to the JD prompt (`input/Questions for the Hiring Manager _attach,ent prompt 1.docx`; verbatim source: parent `context/agent-prompts-import-2026-07-18.md`). Supersedes v1 (themes A–I, 2026-07-11). This questionnaire is the workflow the JD agent executes — every question below must end the session with a coverage status.

**Coverage statuses:** `answered` · `inferred` (credited from something the HM said elsewhere — confirm briefly) · `unknown` · `not_yet_decided` · `skipped` (HM explicitly declined).

**Section handling tags:** `[public]` may inform JD text directly · `[internal]` informs tone, screening and downstream agents but is never quoted in the JD · `[mixed]` per-question judgment (candor answers stay internal).

**Conversation rule (Dana, 2026-07-18):** do not interrogate one-by-one. Bundle naturally related questions from the same section, credit answers given in passing, and keep it a conversation — but the coverage record at the end must account for every question ID.


## 1. Business Context & Purpose  `[public]`

| # | Question |
|---|---|
| 1.1 | Please describe the role and its purpose in your own words. |
| 1.2 | Why does this role exist? |
| 1.3 | Is this a new position or a replacement? |
| 1.4 | If it is a replacement, what worked well in the previous setup or with the previous person? What did not? |
| 1.5 | If it is a new role, what prompted the need? (e.g., business growth, new initiative, restructuring, new capability) |
| 1.6 | What business problem is this role expected to solve? |
| 1.7 | Why is solving this problem important now? |
| 1.8 | What would be the impact if this role remained unfilled? |
| 1.9 | Which business priorities, strategic initiatives, or OKRs does this role directly support? |
| 1.10 | How does this role contribute to your team's objectives and the company's broader strategy? |
| 1.11 | Can you share examples of how similar roles have created value in the past? |
